# Browse AI Ruby Client

A Ruby wrapper for the Browse AI API.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'browse_ai'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install browse_ai

## Documentation
https://www.browse.ai/docs/api/v2
